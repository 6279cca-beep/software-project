## 1. 프로젝트 명
**데이터로 읽는 정치: 선거제도에 따른 여성 의원 비율 비교 분석 도구**
* **한 문장 설명**: 파이썬의 데이터 분석 라이브러리를 활용하여 선거제도에 따른 여성 의원 비율을 통계적으로 검정하고 분석하는 소프트웨어.

## 2. 기획 의도
* **배경:** 세계 각국에서는 소수자의 정치적 대표성을 보장하기 위해서 다양한 선거제도를 채택하고 있음. 기존의 남성 중심사회에서 벗어나면서 여성들의 목소리가 높아지고, 중요해지고 있는 만큼 인구의 반을 차지하는 여성들의 정치적 대표성을 획득하는 것이 필수과제가 됨. 하지만 2025년 기준, 우리나라의 여성의원의 비율은 20.3%로, OECD 38개 회원국 중 하위권인 35위임. 따라서 파이썬을 통해 다양한 나라의 선거제도와 여성의원 비율을 데이터화하여 비교해봄으로써 이러한 제도가 실제로 여성의 정치적 대표성에 유의미한 영향을 미치는지를 확인하고, 우리나라의 정치 발전을 위해서는 어떤 제도가 필요할지를 탐구해보고자 함.

* **추가 이유**: 본인은 앞으로 연론조사 및 리서치 기관에서 정치 분야의 데이터 분석가가 되고 싶음. 따라서 미리 파이썬으로 데이터 분석을 하면서 직업적 역량을 키우고 싶음. 뿐만 아니라 이 소프트웨어는 다른 주제를 분석하는 데도 활용할 수 있을 것이므로 추후에 다른 데이터를 더 분석하면서 실력을 기르고자 함.

* **기획 의도:** 파이썬의 데이터 분석 라이브러리인 `Pandas`와 통계 라이브러리인 `Scipy`를 활용하여 한국, 독일, 호주 등 주요국의 선거 데이터를 분석함. 이를 통해 선거제도의 비례성과 여성 의원 비율 사이의 상관관계를 통계적으로 증명하고 시각화하는 분석 소프트웨어를 구현하고자 함.

## 4. 핵심 기능 및 작동 알고리즘
이 프로그램은 데이터를 로드하고 통계 분석을 수행한 후, 그 결과를 시각화함.

### 핵심 기능
1. **데이터 관리:** 엑셀 CSV 파일로부터 국가별 선거제도 유형(다수제/비례제), 비례의석 비중, 여성 의원 비율 데이터 등을 불러오고 관리함.
2. **기술 통계 분석:** 각 선거제도 그룹별 여성 의원 비율의 평균, 중앙값, 표준편차를 계산하여 요약 보고서 출력.
3. **가설 검정 및 상관 분석:** 두 집단 간 평균 차이 검정(t-test) 수행 및 회귀분석 진행
   - 비례성 정도와 여성 의원 비율 간의 상관계수($r$) 계산.
4. **결과 시각화 (Matplotlib):** 분석 결과를 도표와 그래프로 시각화함.

## 4. 사용할 파이썬 개념
* **Pandas:** 데이터 관리 및 처리.
* **Scipy.stats:** 상관분석 및 가설 검정을 위한 통계 알고리즘 활용.
* **Matplotlib:** 분석 결과를 도표와 그래프로 시각화.
* **Streamlit:** 웹 기반 사용자 인터페이스 구축 및 대시보드 구현.
* **사용자 정의 함수 (`def`):** 데이터 분석 로직과 시각화 로직을 함수로 관리.
* **반복문 및 조건문:** 실행 메뉴 구현.
